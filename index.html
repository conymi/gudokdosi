구독도시

여행보다 깊게, 삶은 넓게

서비스 소개.

구독도시는 지방 소멸 위기 도시와 도시 생활자를 연결하는 라이프스타일 플랫폼입니다. 5축 매칭 시스템으로 사용자의 취향에 맞는 도시를 추천하고, 하루살이·체험·살아보기로 이어지는 관계를 설계합니다. 관계인구 데이터를 지자체에 제공하여 인구 소멸 대응 정책을 지원합니다.

🎯

5축 매칭

템포·소비·공간·관계·생활기반 — 다섯 가지 축으로 사용자와 도시를 매칭합니다.

🏙️

도시 큐레이션

관광공사 데이터 + 로컬 큐레이션으로 관광객이 모르는 진짜 도시를 보여줍니다.

📊

관계인구 데이터

방문·체류·재방문 데이터를 분석하여 지자체 인구 정책에 활용합니다.

앱 경험 흐름.

1
도시가 먼저 질문합니다 — 소개팅처럼, 도시의 페르소나가 사용자에게 질문하며 5축 프로파일을 완성합니다.
2
맞춤 도시 매칭 — 프로파일 기반으로 가장 잘 맞는 도시를 추천합니다.
3
하루살이 코스 — 카카오맵 로컬 검색 API로 검증된 장소 데이터 기반, 맞춤 데이트 코스를 생성합니다.
4
관계 유지 — 방문 후에도 도시와의 관계를 이어갑니다.

카카오맵 API 활용.

🗺️ 로컬 검색 API 사용 목적

한국관광공사 TourAPI로 수집한 장소 데이터를 카카오맵 로컬 검색 API로 크로스체크하여 영업 여부, 카테고리 정확도, 위치 정보를 검증합니다. 검증된 데이터로 사용자에게 신뢰할 수 있는 장소 정보와 코스를 제공합니다.

파일럿 도시: 속초시 (164개 장소 검증 완료)
확장 대상: 공주시, 군산시, 통영시, 서귀포시

기술 스택.

React Native (Expo) 한국관광공사 TourAPI 카카오맵 로컬 검색 API 네이버 로컬 검색 API Python 데이터 분석
